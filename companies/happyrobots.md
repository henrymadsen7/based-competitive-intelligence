# HappyRobots — Competitive Intelligence
## Date: 2026-04-20

### Core Value Prop
"Train for Trust, Measure for Scale" — Building trust infrastructure for enterprises working with LLMs through training and evaluation. NOT a workflow automation platform.

### Key Features
- PRISM: No-code, multi-model workspace for building, running, and scoring evaluations
- WORK: Expert-judged leaderboards on real tasks with Capability Cards and Failure-Mode Library
- COMPASS: 12-week custom enterprise benchmark
- Services: 15-week training-led consulting (Identify → Educate → Develop → Contextualize → Integrate)

### Pricing
Not transparent. PRISM offers "14-day free trial, no credit card required." Enterprise sales model.

### Target Audience
Enterprises needing AI governance, evaluation frameworks, and workforce upskilling.

### Tech Stack Signals
- Multi-model evaluation platform
- PRISM hosted at prism.happyrobots.com
- SSO integration mentioned

### UX/UI Patterns
- No-code visual workspace for evaluation building (drag-and-drop)
- Clean, professional enterprise aesthetic
- Heavy emphasis on methodology and frameworks

### Integration Ecosystem
PRISM supports multiple models for comparison. SSO integration.

### Competitive Weaknesses/Gaps
- NOT a no-code automation/workflow builder — different category entirely
- No transparent pricing
- Heavy consulting/services component (15-week program)
- Niche focus on trust/evaluation rather than end-user automation

### Reverse-Engineering Notes
HappyRobots is adjacent, not direct competition. But their trust/evaluation layer is something we should embed into our platform:
- Built-in agent evaluation and benchmarking
- Failure-mode tracking
- Human feedback loops for agent improvement

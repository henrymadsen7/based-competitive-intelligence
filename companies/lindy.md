# Lindy.ai — Competitive Intelligence
## Date: 2026-04-20

### Core Value Proposition
"The ultimate AI assistant for work." Proactive inbox, meeting, and calendar management via iMessage/SMS. Claims to give users "2 hours back every day."

### Key Features
- Ask/Act/Anticipate framework (search across tools, execute tasks, proactive alerts)
- iMessage-native interface (primary UX is texting your AI assistant)
- Meeting recording and automatic note-taking
- Inbox management (drafts replies in user's voice)
- Meeting scheduling, prep, and follow-up
- Learns style/preferences over time ("memories")
- 7-day free trial, 60-second onboarding
- Hundreds of app integrations

### Pricing
- Plus: $49.99/mo (standard usage, 2 inboxes)
- Pro: $99.99/mo (3x usage, 3 inboxes, computer use)
- Max: $199.99/mo (7x usage, 5 inboxes, more computer use)
- Enterprise: Custom (SSO, SCIM, audit logs, dedicated support)

### Target Audience
Professionals, executives, directors, knowledge workers overwhelmed by email/scheduling admin.

### Tech Stack Signals
- SF-based, in-office 4 days/week
- iMessage/SMS integration (Apple Messages framework)
- SOC 2, HIPAA, GDPR, PIPEDA compliant
- "Autonomous agents that can reason, plan, and execute complex tasks"

### UX/UI Patterns
- Natural language / text-first — no visual workflow builder
- Chat interface optimized for iMessage; SMS fallback for Android
- Homepage emphasizes testimonials ("Wall of Love") from influencers
- Clean, minimal landing page with strong social proof

### Integration Ecosystem
100+ integrations: Gmail, Outlook, Google Calendar, Slack, Notion, CRMs. Categorized: Personal Assistant, Sales, Marketing, Customer Support.

### Competitive Weaknesses/Gaps
- iOS-centric (Android gets SMS, not iMessage experience)
- No transparent API or self-hosted option
- Approval-gated sending limits full autonomy
- Premium pricing may limit SMB adoption
- No visual workflow builder — purely conversational
- No team collaboration features visible

### Reverse-Engineering Notes
Lindy owns the "AI executive assistant via iMessage" niche. To beat them:
- Multi-platform native (iMessage, WhatsApp, Slack, Telegram, Web)
- Visual workflow editor behind the conversational layer
- Self-hosted option for privacy-conscious enterprises
- Team shared agents (not just individual)
- Transparent pricing without inbox limits

# Pentagon.run — Competitive Intelligence
## Date: 2026-04-20

### Core Value Prop
"Stop running agents. Start running teams." Workspace for AI employees to communicate, coordinate, and get work done as a team.

### Key Features
- Spatial Canvas: visual workspace where every agent has a place
- Teams: group agents by function (Engineering, ops, research)
- Channels: structured communication between agents
- Group Chats: agents create group conversations and self-organize
- Reports & Status: each agent maintains live status report
- Task Management: agents create, track, and complete tasks
- Agent Persistence: conversations, decisions, tribal knowledge compounds over time
- Human-in-the-loop: agents report back with live status updates
- Access control: granular permissions at folder, tool, action level
- Visible coordination: every conversation and decision is auditable
- Sandboxing: hardware-level isolation with lightweight VMs (coming soon)

### Pricing
Not publicly disclosed. macOS app download only. "Talk to us" for enterprise.

### Target Audience
Engineering teams, AI-native organizations wanting to run multiple agents as a team.

### Tech Stack Signals
- YC P26 batch
- macOS app (Apple Silicon M1+, macOS 14+, 8GB RAM, 200MB disk)
- Supports Claude, Codex soon
- 6,000+ persistent agents, 900,000+ messages sent, 4,800+ tasks completed, 20:1 agent-to-human ratio

### UX/UI Patterns
- Desktop app (not web-first)
- Slack-like interface for agent teams
- Visual spatial canvas
- Message-based agent coordination
- Real-time status reports

### Integration Ecosystem
Claude, Codex (soon). Likely file system, GitHub, code repos.

### Competitive Weaknesses/Gaps
- macOS only (no Windows, no Linux, no web)
- No pricing transparency
- Very early stage
- Desktop app limits accessibility
- No visual workflow builder
- No SaaS integration ecosystem visible

### Reverse-Engineering Notes
Pentagon owns the "agent team workspace" concept.
To beat them:
- Cross-platform (web, macOS, Windows, Linux, mobile)
- Built-in SaaS integrations (not just code repos)
- Visual workflow builder + chat workspace combined
- Cloud-first with local sync option
- Transparent pricing and self-serve

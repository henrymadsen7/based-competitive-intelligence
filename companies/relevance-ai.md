# Relevance AI — Competitive Intelligence
## Date: 2026-04-20

### Core Value Proposition
"The home of the AI Workforce." Focused on GTM (Go-To-Market) teams. Build agents and multi-agent teams (Workforces) that autonomously handle sales, marketing, support, ops.

### Key Features
- No-code agent builder (Prompt, Tools, Knowledge, Triggers, Evals, Settings tabs)
- Multi-agent "Workforce" builder with visual canvas
- Pre-built agent templates: BDR Agent, Research Agent, Inbound Qualification, Customer Support
- 2,000+ integrations
- AI adoption roadmap: L1 Assisted → L2 Copilot → L3 Autopilot → L4 Self-Driving
- SuperGTM chat interface (delegate tasks via natural language)
- Knowledge base connections (Google Drive, SharePoint, Notion, websites)
- Human-in-the-loop: smart escalations, approvals
- A/B testing for agents
- Analytics dashboard
- Activity center / monitoring
- Schedule tasks
- Calling & meeting agents (Team+)
- Version control on agents
- "Programmatic GTM" plugin for Claude Code & Codex
- Custom LLM bring-your-own-model
- SOC 2 Type II, GDPR, SSO, RBAC, data residency

### Pricing
- Free: $0/mo, 200 actions/mo + $2 vendor credits, 1 user, 1 project, 1 workforce
- Pro: $19/mo (annual), 30,000 actions/yr + $240 vendor credits/yr, 2 build users
- Team: $234/mo (annual), 84,000 actions/yr + $840 vendor credits/yr, 5 build users, 45 end users
- Enterprise: Custom, unlimited users/projects, agent evaluations, work hour controls

### Target Audience
Sales, marketing, customer success, GTM operations teams at B2B companies.
Customers: Canva, Autodesk, KPMG, Lightspeed Commerce, Qualified, Send Payments.

### Tech Stack Signals
- React-based marketing site
- Offices in Sydney, San Francisco, Barcelona
- Supports Claude Sonnet, GPT models
- GitHub presence (RelevanceAI org)
- Discord community

### UX/UI Patterns
- Tabbed agent builder interface (Prompt, Tools, Knowledge, Triggers, Evals, Settings)
- Visual canvas for multi-agent workforce orchestration
- Chat-first delegation ("Ask Relly anything")
- Template marketplace for cloning pre-built agents
- Build/Run/Monitor toggle for agents

### Integration Ecosystem
1,000+ apps connected. CRMs, email, calendars, web scraping, enrichment tools, communication platforms.

### Competitive Weaknesses/Gaps
- Pricing escalates quickly ($234/mo for Team tier)
- Action-based pricing is opaque (what counts as an "action"?)
- Vendor credits system adds complexity
- Heavy GTM/Sales focus means less flexibility for non-GTM use cases
- Confusing brand architecture (SuperGTM, Relevance AI, Workforces, Agents, Tools all overlap)
- No self-host option

### Reverse-Engineering Notes
Relevance AI demonstrates "AI Workforce" branding and multi-agent teams resonate in GTM.
To beat them:
- Flat, predictable pricing (no action credits, no vendor credits)
- General-purpose platform (not just GTM)
- Simpler UX — fewer tabs, more conversational
- Native code extensibility
- Bring-your-own-cloud deployment

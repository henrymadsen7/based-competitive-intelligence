# GetModern.ai — Competitive Intelligence
## Date: 2026-04-20

### Core Value Prop
"AI agents for IT, HR & Operations teams." Agents take tickets from request to resolution across IT, HR, Legal, Finance. 50% closed in 30 days. 70% by day 60.

### Key Features
- Service desk automation (access requests, provisioning, onboarding, offboarding)
- AI-native workflows (natural language description → built workflow → approve → run)
- Access management (just-in-time access, birthright provisioning, automatic offboarding)
- Policy enforcement (every action checked against policy, automatic audit trail)
- Learns with usage (learns from manual fixes, suggests automations, coverage compounds)
- Modern Managed: full platform + managed service desk team
- Integrations: ServiceNow, Zendesk, Datadog, Microsoft Graph, Google Workspace, Workday, BambooHR, SAP, Salesforce, Atlassian, Slack, Cisco Meraki

### Pricing
Pricing page shows "Coming soon" — enterprise sales model only.

### Target Audience
Enterprise IT teams replacing legacy ITSM (ServiceNow, Jira SM, Freshservice).

### Tech Stack Signals
- YC-backed
- Compares itself to ServiceNow, Jira SM, Freshservice
- SOC 2 Type 2, SCORM, GDPR, ISO 27001, SAML SSO, audit logging

### UX/UI Patterns
- Natural language workflow creation
- Policy-gated automation
- Enterprise comparison pages
- Managed service overlay option

### Integration Ecosystem
ServiceNow, Zendesk, Datadog, Microsoft Graph, Google Workspace, Workday, BambooHR, SAP, Salesforce, Atlassian, Slack, Cisco Meraki.

### Competitive Weaknesses/Gaps
- No transparent pricing
- Narrow focus (ITSM/HR only)
- Heavy enterprise sales motion
- No self-serve onboarding visible
- No mention of custom API integrations beyond listed apps

### Reverse-Engineering Notes
Modern proves AI-native ITSM is a viable enterprise category.
To generalize and beat them:
- ITSM as ONE vertical template among many (finance, legal, sales, ops)
- Self-serve onboarding with instant value
- Transparent pricing
- Custom API integration builder (not just pre-built connectors)
